Sobre o Caudais

Esta aplicação web foi desenvolvida como parte dos projetos finais de curso das alunas Nicole Arquissandas (Licenciatura em Engenharia Informática) e Mariana Herculano (Licenciatura em Engenharia Informática de Gestão), da Universidade Lusófona.

O objetivo do projeto é proporcionar uma plataforma interativa para a visualização, análise estatística e reconstrução de dados de caudal em sistemas de distribuição de água, com aplicação prática no contexto da gestão e monitorização de redes hidráulicas.

A aplicação permite o carregamento de ficheiros com dados de caudal, visualização geográfica através de mapas interativos, exploração de séries temporais com gráficos dinâmicos e comparação de diferentes cenários antes e depois de tratamento estatístico.

Este trabalho foi desenvolvido em estreita colaboração com o LNEC – Laboratório Nacional de Engenharia Civil, que forneceu dados reais e orientações técnicas essenciais para a sua concretização.

Laboratório Nacional de Engenharia Civil

O LNEC é uma instituição pública de investigação reconhecida nacional e internacionalmente, com uma longa tradição na produção de conhecimento científico e tecnológico nas áreas da engenharia civil e do ambiente.

No âmbito deste projeto, o LNEC colaborou ativamente através da disponibilização de dados reais de caudal, bem como do acompanhamento técnico e científico necessário para garantir a relevância e aplicabilidade prática da aplicação desenvolvida.

Esta parceria permitiu que a aplicação respondesse a desafios reais na monitorização e análise de sistemas de distribuição de água, reforçando o seu valor enquanto ferramenta de apoio à investigação e à gestão de infraestruturas hidráulicas.

Tecnologias Utilizadas

Backend

  • Python 3
    Linguagem principal usada para o back-end
  • Django
    Framework web baseada no padrão MVT (Model-View-Template)
  • RPy2
    Biblioteca que permite executar scripts R diretamente no Python
  • Pandas & NumPy
    Para processamento e análise de séries temporais
  • SQLite
    Base de dados utilizada
  • Script R
    com métodos JQ e TBATS importados do TFC anterior e integrados ao projeto com adaptações

Frontend

  • HTML5
    Para estruturação das páginas nos templates Django
  • CSS3
    Para estilização moderna e responsiva
  • JavaScript
    Para interatividade (ex: gráficos, ações no menu)
  • Chart.js
    Biblioteca JS usada para visualização de gráficos interativos
  • Leaflet.js
    Para visualização de mapas interativos
